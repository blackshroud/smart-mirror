Smart Mirror

This project is inspired by HomeMirror and Michael Teeuw's Magic Mirror and evancohens SmartMirror . It uses annyang for voice interactivity, electron to make it cross platform, and integrates with Philips Hue and can also display RSS feeds and iCal for your convenience.

Getting Started

Hardware Components

  • Raspberry Pi 2 (optional)
  • USB Microphone (Or Webcam w/ microphone)
  • Monitor (with the bezel removed - optional)
  • Mirror Pane (aka Observation Glass) (optional)
  • Philips Hue (optional)

Installation

I have set this up so that it can run of of any web server, so as long as you have a webserver running somewhere, that you can host this from, you should be fine. (keeping in mind that you MUST use Chrome to display it)

Setting up the configuration

Done? Excellent, let's continue.

Time to update the config file... You'll need to fill in things in js/config.js:

  1. A Forecast API key (don't worry, it's free)
  2. Philips Hue Bridge IP address with a configured user. Details about how to set this up in the Philips Hue Developer Documentation
  3. Set RSS feed location in config.js
  4. Add iCal URL to config.js

To set the default MAP home, update the address located in map-service.js
